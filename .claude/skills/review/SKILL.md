이 skill은 작성된 TIL 문서의 글 품질(구조·명료성·문법)을 리뷰하는 진입점이다.

## 입력 처리

- 인자로 파일 경로가 주어지면 그것을 리뷰 대상으로 한다.
- 인자가 없으면:
  1. IDE에서 현재 열린 파일이 `archive/` 하위라면 그 파일을 사용.
  2. 아니면 "어떤 파일을 리뷰할까요?"라고 묻는다.
- 사용자가 특정 섹션만 리뷰해달라고 하면 그 범위를 agent에 전달.

## 실행 절차

1. **til-reviewer agent를 호출**해 파일 경로(또는 섹션 범위)를 넘긴다. 한국어 응답, CLAUDE.md 컨벤션 기준 점검을 명시.
2. agent가 반환한 우선순위별 코멘트를 사용자에게 그대로 보여준다.
3. 사용자가 적용할 항목을 지정하면(`1, 3, 5 적용해줘` / `상 모두` / `all`) Edit으로 반영.
4. 사실관계가 의심되는 항목이 보이면 사용자에게 `/factcheck` 실행을 권장.

## 출력 형식

agent의 리뷰 결과를 그대로 보여준 뒤, 마지막에:

> 적용할 항목 번호를 알려주세요(예: `상 모두` / `1, 3` / `all` / `skip`).
> 사실관계가 의심되면 `/factcheck`를 별도로 돌리는 것을 권장합니다.

## 주의

- 사용자 승인 없이 본문을 직접 수정하지 않는다.
- 글쓴이의 의도·관점을 바꾸는 첨삭은 하지 않는다(어휘·구조 다듬기에 한정).
- "지적 없음"이라는 결과가 나오면 솔직히 그렇게 전달한다 — 무리하게 흠을 만들지 않는다.
